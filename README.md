## 📖 Sobre o Projeto

**Conversor de Moedas** é muito mais que um simples conversor! 🚀

É uma aplicação web moderna e intuitiva que traz para você as **cotações mais recentes** do mercado financeiro em tempo real. Desenvolvido com tecnologias web fundamentais (HTML, CSS e JavaScript puro), este projeto demonstra como criar uma aplicação completa e funcional sem frameworks complexos.

### 🎯 Por que usar o Conversor de Moedas?

- 🔄 **Sempre Atualizado**: Cotações renovadas automaticamente a cada 5 minutos
- ⚡ **Instantâneo**: Conversão em tempo real enquanto você digita
- 🌐 **Internacional**: Suporte para as principais moedas mundiais
- ₿ **Moderno**: Inclui criptomoedas como Bitcoin
- 📱 **Responsivo**: Funciona perfeitamente em qualquer dispositivo
- 🎨 **Interface Limpa**: Design minimalista e fácil de usar

## ✨ Funcionalidades Principais

| Recurso | Descrição |
|---------|-----------|
| 🔴 **Ao Vivo** | Cotações em tempo real direto da [AwesomeAPI](https://docs.awesomeapi.com.br/api-de-moedas) |
| ⚡ **Dinâmico** | Conversão automática ao digitar ou trocar moedas |
| 🪙 **Multi-Moedas** | Dólar, Euro, Real, Bitcoin e muito mais! |
| 🎨 **Visual Inteligente** | Bandeiras e símbolos mudam automaticamente |
| 🌎 **Formatação Local** | Valores formatados corretamente (R$ 1.000,00 / US$ 1,000.00) |
| 🔄 **Auto-Refresh** | Atualização silenciosa das cotações a cada 5 minutos |

### 📚 O que foi usado:

**🏗️ HTML5**
- Estrutura semântica e acessível
- Formulários otimizados

**🎨 CSS3**
- Design responsivo e mobile-first
- Animações suaves e transições
- Layout flexível com Flexbox

**⚙️ JavaScript ES6+**
- `async/await` para requisições assíncronas
- `Fetch API` para consumo de dados externos
- `Intl.NumberFormat` para formatação monetária internacional
- `setInterval` para atualização automática
- Manipulação avançada do DOM
- Event listeners para interatividade
